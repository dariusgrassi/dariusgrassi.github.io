Projects

ML-Driven Memory System Auto-Tuning University of Texas at Austin, 2024-2025

Developed an automated tuning system for zswap compression parameters using PyTorch and feature engineering. Built a Redis benchmarking pipeline for evaluation and optimal configuration identification. The online learned system significantly reduced the time required to discover optimal configurations compared to traditional search methods.

Programmable NIC Memory Management University of Texas at Austin, 2023-2024

Prototyped a memory management system for programmable NICs to support cloud applications. Implemented various allocation policies using DPDK, a kernel-bypass framework, enabling high-throughput packet processing.

FPGA Acceleration Framework University of Texas at Austin, 2023

Developed a High-Level Synthesis (HLS) implementation of a CNN accelerator, including performance and resource utilization analysis. Also designed optical flow processing accelerators using HLS for an FPGA accelerator streaming platform.

Software Network Switch with RDMA University of Texas at Austin, 2022

Built a custom software switch using DPDK for kernel-bypass processing on a 100 Gbps testbed. Integrated RDMA for high-speed communication and conducted a thorough performance evaluation of TCP and RDMA trade-offs.
